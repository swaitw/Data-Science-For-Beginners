# Cuba di Excel

## Arahan

Tahukah anda bahawa anda boleh mencipta carta donat, pai, dan wafel dalam Excel? Dengan menggunakan set data pilihan anda, cipta ketiga-tiga carta ini terus dalam hamparan Excel.
