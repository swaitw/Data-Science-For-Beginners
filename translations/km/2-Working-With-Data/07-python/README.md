бһҒбһҺбҹҲбһҠбҹӮбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҹбҹҶбһҺбһҪбһҡбһҸбһ¶бһҳбһ—бһ¶бһҹбһ¶бһҹбҹҶбһҺбһҪбһҡ бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһ”бһҸбҹӢбһ”бҹӮбһ“бһ”бҹҶбһ•бһ»бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбһәбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһ¶бһ…бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҮбһ¶бһ„бҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҒбҹ’бһӣбҹҮбҹ— бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶бһ„бһ“бҹҒбҹҮ бһңбһ¶бһҳбһ·бһ“бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҸбһ¶бһҳбһҡбһҷбҹҲ SQL бһ‘бҹҒбҹ”
бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҺбһ¶бһҳбһҪбһҷбһҖбҹҸбһҠбҹ„бһҷ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ¶бһ“бһ—бһ¶бһҹбһ¶бһҡбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһўбҹ’бһ“бһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ—бһ¶бһҹбһ¶бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳбһ—бһ¶бһҹбһ¶бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

* **[Python](https://www.python.org/)** бһҮбһ¶бһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһ‘бһјбһ‘бҹ… бһҠбҹӮбһӣбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһӮбҹҒбһ…бһ¶бһҸбҹӢбһ‘бһ»бһҖбһҗбһ¶бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһ—бһ¶бһ–бһҹбһ¶бһҳбһүбҹ’бһүбһҡбһ”бһҹбҹӢбһңбһ¶бҹ” Python бһҳбһ¶бһ“бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ“бһҪбһҷбһҠбһӣбҹӢбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ‘бһјбһ‘бҹ…бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһҠбһҖбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһ бһ¶бһҡбһёбһӣ ZIP бһ¬бһ”бһҳбҹ’бһӣбҹӮбһ„бһҡбһјбһ”бһ—бһ¶бһ–бһ‘бҹ…бһҮбһ¶бһ–бһҺбҹҢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹ’бһҡбһ„бҹӢбҹ” бһӣбһҫбһҹбһ–бһёбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ Python бһҖбҹҸбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһңбһ·бһңбһҢбҹ’бһҚбһңбҹҒбһ”бһҹбһ¶бһҷбһ•бһ„бһҠбҹӮбһҡбҹ”
* **[R](https://www.r-project.org/)** бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ»бһҡбһ¶бһҺбһҳбһҪбһҷбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹ’бһҗбһ·бһҸбһ·бҹ” бһңбһ¶бһҖбҹҸбһҳбһ¶бһ“бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ’бҹҶбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҸбһ»бһҖ (CRAN) бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһңбһ¶бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһҷбҹүбһ¶бһ„бһҺбһ¶ R бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһјбһ‘бҹ…бһ‘бҹҒ бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ‘бһјбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҠбҹӮбһ“бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”
* **[Julia](https://julialang.org/)** бһҮбһ¶бһ—бһ¶бһҹбһ¶бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһҗбһ¶бһ“бһ№бһ„бһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹӮбһӣбһӣбҹ’бһўбһҮбһ¶бһ„ Python бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһңбһ¶бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһЎбһ„бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ Python бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ¶бһҳбһүбҹ’бһүбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҮбһ¶бһҳбһҪбһҷбһ—бһ¶бһҹбһ¶бһ“бҹҒбҹҮбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹҸбһҮбҹ’бһҡбһ¶бһӣбһҮбҹ’бһҡбҹ…бһўбҹҶбһ–бһё Python бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҷбҹ„бһ„бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ’бһ“бһ’бһ¶бһ“бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

бһҷбһҫбһ„бһ“бһ№бһ„бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ…бҹҶбһ“бһҪбһ“бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһўбҹ„бһҷбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҹбҹ’бһҸбһёбһ–бһёбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бҹ” бһңбһ¶бһ“бһ№бһ„бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“ бһ бһҫбһҷбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһ–бһёбһҖбһ“бҹ’бһӣбҹӮбһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ–бһҪбһҖбһңбһ¶бҹ”

> **бһҠбҹҶбһ”бһјбһ“бҹ’бһҳбһ¶бһ“бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ”бҹҶбһ•бһ»бһҸ**бҹ” бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһҺбһ¶бһҳбһҪбһҷбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҠбһ№бһ„бһҗбһ¶бһҸбһҫбһ’бҹ’бһңбһҫбһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ… бһҹбһјбһҳбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҸбһ¶бһҳбһўбҹҠбһёбһ“бһ’бһҫбһҺбһ·бһҸбҹ” [Stackoverflow](https://stackoverflow.com/) бһҮбһ¶бһ‘бһјбһ‘бҹ…бһҳбһ¶бһ“бһӮбҹҶбһҡбһјбһҖбһјбһҠ Python бһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹҒбһҹбһҖбһҖбһҳбҹ’бһҳбһ‘бһјбһ‘бҹ…бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ”



## [бһҹбҹҶбһҺбһҪбһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһ»бһ“бһҳбҹҒбһҡбҹҖбһ“](https://ff-quizzes.netlify.app/en/ds/quiz/12)

## бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбһ¶бһҡбһ¶бһ„ бһ“бһ·бһ„ Dataframes

бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбҹ’бһӮбһ¶бһӣбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбһ¶бһҡбһ¶бһ„бһ бһҫбһҷбһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһ“бһ·бһҷбһ¶бһҷбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бҹ” бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹ’бһҡбһҫбһ“ бһ бһҫбһҷбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбһ“бҹ…бһҸбһ¶бһҡбһ¶бһ„бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶ бһңбһ¶бһ–бһ·бһҸбһҮбһ¶бһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ SQL бһҡбһҪбһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһңбһ¶бҹ” бһ‘бҹ„бҹҮбһҷбҹүбһ¶бһ„бһҺбһ¶ бһҳбһ¶бһ“бһҖбһҡбһҺбһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһўбһ¶бһӣбһҠбҹӮбһӣбһҷбһҫбһ„бһҳбһ¶бһ“бһҸбһ¶бһҡбһ¶бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ¬бһңбһ·бһ…бһ¶бһҡбһҺбһ¶бһўбҹҶбһ–бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷ бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһҠбһҫбһҳбҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҳбһ¶бһ“бһҖбһҡбһҺбһёбһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡ бһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһҫбһҳ бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһңбһ·бһ…бһ·бһҸбҹ’бһҡбһҹбһ·бһӣбҹ’бһ”бҹҲбҹ” бһ–бһёбһҡбһҮбҹҶбһ бһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Pythonбҹ”

бһҳбһ¶бһ“бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Python бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҒбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбһ¶бһҡбһ¶бһ„бҹ–
* **[Pandas](https://pandas.pydata.org/)** бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһўбҹ’бһңбһёбһҠбҹӮбһӣбһӮбҹҒбһ бҹ…бһҗбһ¶ **Dataframes** бһҠбҹӮбһӣбһҹбҹ’бһҡбһҠбҹҖбһ„бһ‘бҹ…бһ“бһ№бһ„бһҸбһ¶бһҡбһ¶бһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһ¶бһ“бһҮбһҪбһҡбһҲбһҡвҖӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮ бһ бһҫбһҷбһўбһ¶бһ…бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бһӣбһҫбһҷбҹ„бһ„бһ‘бҹ…бһӣбһҫбһҮбһҪбһҡвҖӢ бһҮбһҪбһҡбһҲбһҡ бһ“бһ·бһ„ Dataframes бһҮбһ¶бһ‘бһјбһ‘бҹ…бҹ”
* **[Numpy](https://numpy.org/)** бһӮбһәбһҮбһ¶бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ **tensors**, бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ **arrays** бһҜбһҖбһ—бһ¶бһ–бһ…бҹ’бһҡбһҫбһ“бһңбһ·бһҳбһ¶бһҸбҹ’бһҡбҹ” Array бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһ–бһёбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ бһҫбһҷбһңбһ¶бһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбһ¶бһҳбһүбҹ’бһүбһҮбһ¶бһ„ dataframe бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһҮбһ¶бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҡбҹҶбһҒбһ¶бһ“бһҸбһ·бһ…бһҮбһ¶бһ„бҹ”

бһҳбһ¶бһ“бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһҠбһ№бһ„бһҠбһјбһ…бһҮбһ¶бҹ–

* **[SciPy](https://www.scipy.org/)** бһӮбһәбһҮбһ¶бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһҒбҹ’бһӣбҹҮбҹ” бһҷбһҫбһ„бһ”бһ¶бһ“бһҹбҹ’бһӮбһ¶бһӣбҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ“бҹҒбҹҮбһҡбһҪбһ…бһ–бҹҒбһӣбһ“бһ·бһҷбһ¶бһҷбһ–бһёбһ”бҹ’бһҡбһ—бһ¶бһ–бһ“бһ№бһ„бһҹбҹ’бһҗбһ·бһҸбһ·

### Series 

**Series** бһӮбһәбһҮбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһҳбҹ’бһӣбҹғбһҳбһҪбһҷбһҮбһ¶бһҹбҹҒбһҡбһё бһңбһ¶бһҹбҹ’бһҡбһҠбҹҖбһ„бһ“бһ№бһ„бһ”бһүбҹ’бһҮбһё бһ¬ numpy arrayбҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҳбһ¶бһҗбһ¶бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһӣбһҫбһҹбҹҶбһҒбһ¶бһ“бҹӢбһӮбһә series бһҳбһ¶бһ“ **index** бһ•бһ„бһҠбҹӮбһҡ бһ бһҫбһҷбһ–бҹҒбһӣбһҷбһҫбһ„бһ”бҹҶбһ–бҹҒбһүбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһӣбһҫ series (бһ§. бһ”бһјбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ) бһңбһ¶бһ“бһ№бһ„бһӮбһ·бһҸбһҒбҹ’бһңбҹҮбһ“бҹғ index бҹ” Index бһўбһ¶бһ…бһҮбһ¶бһҗбҹ’бһ“бһ¶бһҖбҹӢбһӣбҹҒбһҒбһҒбҹ’бһҹбҹӮбһӮбҹҶбһ“бһҡбһҠбҹӮбһӣбһҹбһ¶бһҳбһүбҹ’бһү (бһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸ series бһ–бһёбһ”бһүбҹ’бһҮбһё бһ¬ array бһңбһ¶бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҮбһ¶бһӣбҹҶбһ“бһ¶бҹҶбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’) бһ¬бһўбһ¶бһ…бһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҠбһјбһ…бһҮбһ¶ бһӣбҹҶбһ“бһ¶бҹҶбһҖбһ¶бһӣбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘бҹ”

> **бһҖбһҸбҹӢбһ…бҹҶбһҺбһ¶бҹҶ**бҹ– бһҳбһ¶бһ“бһҖбһјбһҠ Pandas бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһҠбһ¶бһҹбһҹбһҳбҹ’бһ—бһ¶бһҡбҹҲбһҮбһ¶бһҳбһҪбһҷ [`notebook.ipynb`](notebook.ipynb)бҹ” бһҷбһҫбһ„бһ”бһ¶бһ“бһӮбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ бһ бһҫбһҷбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҖбһҳбһҫбһӣбһҖбҹ’бһҡбһҠбһ¶бһҹбһ–бҹҒбһүбһӣбҹҒбһүбһ”бһ¶бһ“бҹ”

бһҸбһҫбһўбҹ’бһ“бһҖбһӮбһ·бһҸбһҗбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһҪбһҷбҹ– бһҷбһҫбһ„бһ…бһ„бҹӢбһңбһ·бһ—бһ¶бһӮбһҖбһ¶бһҡбһӣбһҖбҹӢбһ‘бҹҶбһ“бһ·бһүбһҖбһ“бҹ’бһӣбҹӮбһ„бһӣбһҖбҹӢбһҖбһҳбҹ’бһўбһ·бһҸбһ‘бһ№бһҖбһҖбһҖбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ” бһҷбһҫбһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһҪбһҡбһӣбҹҒбһҒбһӣбһҖбҹӢ (бһ…бҹҶбһ“бһҪбһ“бһ‘бҹҶбһ“бһ·бһүбһҠбҹӮбһӣбһ”бһ¶бһ“бһӣбһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҗбҹ’бһ„бҹғ) бһҡбһҷбҹҲбһ–бҹҒбһӣбһҒбҹ’бһӣбҹҮбҹ–

```python
start_date = "Jan 1, 2020"
end_date = "Mar 31, 2020"
idx = pd.date_range(start_date,end_date)
print(f"Length of index is {len(idx)}")
items_sold = pd.Series(np.random.randint(25,50,size=len(idx)),index=idx)
items_sold.plot()
```

```python
additional_items = pd.Series(10,index=pd.date_range(start_date,end_date,freq="W"))
```

бһ–бҹҒбһӣбһҷбһҫбһ„бһ”бһјбһҖ series бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶ бһҷбһҫбһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ…бҹҶбһ“бһҪбһ“бһҹбһҡбһ»бһ”бҹ–
```python
total_items = items_sold.add(additional_items,fill_value=0)
```

> **бһҖбһҸбҹӢбһ…бҹҶбһҺбһ¶бҹҶ** бһҷбһҫбһ„бһҳбһ·бһ“бһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶SimpleбҹҚ `total_items+additional_items` бһҠбҹӮбһӣбһҹбһ¶бһҳбһүбҹ’бһүбһ‘бҹҒбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҷбһҫбһ„бһ’бҹ’бһңбһҫбһ“бҹ„бҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҸбһҳбҹ’бһӣбҹғ `NaN` (бһӮбҹ’бһҳбһ¶бһ“бһӣбҹҒбһҒ) бһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ series бһ•бһӣбһ·бһҸбһ–бһёбһҖбһ¶бһҡбһ”бһјбһҖбҹ” бһ“бҹҒбҹҮбһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбһҳбҹ’бһӣбҹғбһҒбҹ’бһңбҹҮбһҒбһ¶бһҸбһ“бҹ…бһ…бҹҶбһҺбһ»бһ… index бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҖбҹ’бһ“бһ»бһ„ series `additional_items` бһ бһҫбһҷбһҖбһ¶бһҡбһ”бһјбһҖ `NaN` бһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһңбһёбҹ—бһҠбһ‘бҹғбһҹбҹ’бһ‘бһҫбһҡбһҸбҹӮбһ•бҹ’бһҸбһӣбҹӢ `NaN`бҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ•бҹ’бһ‘бһ»бһҖбһҸбһҳбҹ’бһӣбҹғ `fill_value` бһҒбһҺбҹҲбһ–бҹҒбһӣбһ”бһјбһҖбҹ”

бһҮбһ¶бһҳбһҪбһҷ series бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ бһҷбһҫбһ„бһўбһ¶бһ…бһ”бҹ’бһҠбһјбһҡбһҹбҹҶбһҺбһ»бҹҶ (resample) бһҮбһ¶бһҳбһҪбһҷбһңбҹҒбһӣбһ¶бһӣбһҖбһҠбһјбһ…бһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҹбһ“бҹ’бһҳбһҸбһҗбһ¶бһҷбһҫбһ„бһ…бһ„бҹӢбһӮбһҺбһ“бһ¶бһҳбһ’бҹ’бһҷбһҳбһ…бҹҶбһ“бһҪбһ“бһӣбһҖбҹӢбһҡбҹҖбһ„бһҒбҹӮбҹ” бһҷбһҫбһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһӣбһҖбҹ’бһҒбһҺбҹҲбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

**бһҖбһҸбҹӢбһ…бҹҶбһҺбһ¶бҹҶ** бһҷбһҫбһ„бһҖбҹҸбһўбһ¶бһ…бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҸбһ¶бһҡбһ¶бһ„бһ“бҹҒбҹҮбһҠбҹ„бһҷбһ”бһҳбҹ’бһӣбҹӮбһ„бһҸбһ¶бһҡбһ¶бһ„бһҳбһ»бһ“бһ–бһёбһҮбһҪбһҡбһҸбҹ’бһҡбҹҖбһҳбһҮбһҪбһҡбһҲбһҡ бһҠбһјбһ…бһҮбһ¶

бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ `.T` бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„бһҸбһ¶бһҡбһ¶бһ„ DataFrame бһҠбҹ„бһҷбһ”бҹ’бһҸбһјбһҡбһҮбһҪбһҡбһ“бһ·бһ„бһҮбһҪбһҡбһҲбһҡ бһ бһҫбһҷ `rename` бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҷбһҫбһ„бһ”бҹ’бһҸбһјбһҡбһҲбҹ’бһҳбҹ„бҹҮбһҮбһҪбһҡбһҲбһҡбһҸбһ¶бһҳбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһ»бһ“бҹ”

бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҮбһ¶бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹӮбһӣбһҷбһҫбһ„бһўбһ¶бһ…бһ…бһ¶бһҸбҹӢбһ‘бһ»бһҖбһ‘бҹ…бһӣбһҫ DataFramesбҹ–

**бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҮбһҪбһҡбһҲбһҡ**бҹ” бһҷбһҫбһ„бһўбһ¶бһ…бһҮбҹ’бһҡбһҫбһҹбһҮбһҪбһҡбһҲбһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбҹ„бһҷбһҹбһҡбһҹбҹҒбһҡ `df['A']` - бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбҹ’бһҡбһЎбһ”бҹӢбһҸбһҳбҹ’бһӣбҹғбһҮбһ¶ Seriesбҹ” бһҷбһҫбһ„бһҖбҹҸбһўбһ¶бһ…бһҮбҹ’бһҡбһҫбһҹбһҮбһҪбһҡбһҲбһҡбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ‘бҹ… DataFrame бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҠбҹ„бһҷбһҹбһҡбһҹбҹҒбһҡ `df[['B','A']]` - бһңбһ¶бһ”бһҳбҹ’бһӣбҹӮбһ„бһ‘бҹ…бһҮбһ¶ DataFrame бһҳбһҪбһҷбһ•бһ„бһҠбҹӮбһҡбҹ”

**бһҸбһҳбҹ’бһҡбһ„** бһҮбһҪбһҡбһҠбҹ„бһҷбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹ’бһҡбһ¶бһ“бһ…бҹ„бһӣбһҮбһҪбһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҮбһҪбһҡбһҲбһҡ `A` бһҸбһ·бһ…бһҮбһ¶бһ„ бһ¬бһҹбҹ’бһҳбһҫ бҹҘ бһҷбһҫбһ„бһўбһ¶бһ…бһҹбһҡбһҹбҹҒбһҡ `df[df['A']>5]`бҹ”

> **бһҖбһҸбҹӢбһ…бҹҶбһҺбһ¶бҹҶ**бҹ– бһҡбһ”бҹҖбһ”бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ”бҹӮбһҖбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ” бһ”бһ¶бһҡбһ¶бһ„ `df['A']<5` бһ“бһ№бһ„бһҸбҹ’бһҡбһЎбһ”бҹӢбһҡбһ”бһ¶бһҡ boolean бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶ бһҸбһҳбҹ’бһӣбҹғбһҳбһҪбһҷбҹ—бһҖбҹ’бһ“бһ»бһ„ `df['A']` бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғ True бһ¬ False бҹ” бһ–бҹҒбһӣбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫ boolean series бһҮбһ¶ index бһңбһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһЎбһ”бҹӢбһҮбһ¶бһ…бҹҶбһ“бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ DataFrameбҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһҡбһҳбҹ’бһ— boolean code бһҖбҹ’бһ“бһ»бһ„ Python бһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһҡбһҪбһ…бһ”бһ¶бһ“бһ‘бҹҒ бһ бһҫбһҷбһҒбһ»бһҹбһҶбҹ’бһӮбһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһҠбһјбһ…бһҮбһ¶ `df[df['A']>5 and df['A']<7]`бҹ” бһҗбҹӮбһҳбһ‘бһ¶бҹҶбһ„бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһ–бһ·бһҹбҹҒбһҹ `&` бһӣбһҫ boolean series бһҮбһ¶бһҳбһҪбһҷбһҹбһүбҹ’бһүбһ¶бһҹбҹ’бһңбҹҗбһҷбһ”бһ·бһ‘бһҹбһүбҹ’бһүбһ¶ `(` `)` бһҠбһјбһ…бһҮбһ¶ `df[(df['A']>5) & (df['A']<7)]` (бһҹбһүбҹ’бһүбһ¶бһҹбҹ’бһңбҹҗбһҷбһ”бһ·бһ‘бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢ)бҹ”

**бһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһҪбһҡбһҲбһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһҗбҹ’бһҳбһё**бҹ” бһҷбһҫбһ„бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһҪбһҡбһҲбһҡбһҗбҹ’бһҳбһёбһҠбҹӮбһӣбһӮбһҺбһ“бһ¶бһ…бҹҒбһүбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ DataFrame бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бһҡбһ·бһҷбһ¶бһҷбһҹбҹҶбһҺбһҪбһҡбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–
```python
df['DivA'] = df['A']-df['A'].mean() 
``` 
бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһӮбһҺбһ“бһ¶бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„ A бһ“бһ·бһ„бһҸбһҳбҹ’бһӣбҹғбһҳбһ’бҹ’бһҷбһҳбһҡбһ”бһҹбҹӢбһңбһ¶бҹ” бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбһҫбһҸбһЎбһҫбһ„бһ–бһ·бһҸбһҮбһ¶бһӮбһәбһҷбһҫбһ„бһҖбҹҶбһ–бһ»бһ„бһӮбһҺбһ“бһ¶бһҠбҹҶбһҺбһҫбһҡбһҳбһҪбһҷ бһ бһҫбһҷбһҠбһ¶бһҖбҹӢ series бһ“бҹҒбҹҮбһ‘бҹ…бһҒбһ¶бһ„бһҶбҹ’бһңбҹҒбһ„ бһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһҪбһҡбһҲбһҡбһҗбҹ’бһҳбһёбҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһҺбһ¶бһҠбҹӮбһӣбһҳбһ·бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ–бһё series бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһ”бһ¶бһ“бһ‘бҹҒбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһӮбһәбһҒбһ»бһҹбҹ–
```python
# бһҖбһјбһҠбһҒбһ»бһҹ -> df['ADescr'] = "бһ‘бһ¶бһ”" бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ df['A'] < 5 бһ•бҹ’бһ‘бһ»бһҷбһ‘бҹ… "бһҒбҹ’бһ–бһҹбҹӢ"
df['LenB'] = len(df['B']) # <- бһӣбһ‘бҹ’бһ’бһ•бһӣбһҒбһ»бһҹ
``` 
бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮ бһҒбһҺбҹҲбһҠбҹӮбһӣбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҸбһ¶бһҳбһңбҹҒбһҷбҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚ бһҸбҹӮбһ•бҹ’бһҸбһӣбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһҒбһ»бһҹбһҖбҹ’бһ“бһ»бһ„бһ“бҹҗбһҷбһҗбһ¶ бһңбһ¶бһ”бҹӮбһ„бһ…бҹӮбһҖбһ”бҹ’бһҡбһңбҹӮбһ„ series `B` бһ‘бҹ…бһҸбһҳбҹ’бһӣбҹғбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҮбһҪбһҡбһҲбһҡ бһҳбһ·бһ“бһҳбҹӮбһ“бһ”бҹ’бһҡбһңбҹӮбһ„бһ“бҹғбһ’бһ¶бһҸбһ»бһҳбһҪбһҷбҹ—бһҠбһјбһ…бһҠбҹӮбһӣбһҷбһҫбһ„бһ…бһ„бҹӢбһ”бһ¶бһ“бҹ”

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶ expression бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһҠбһјбһ…бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ `apply`бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһўбһ¶бһ…бһҹбһҡбһҹбҹҒбһҡбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–
```python
df['LenB'] = df['B'].apply(lambda x : len(x))
# бһ¬
df['LenB'] = df['B'].apply(len)
```

**бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҮбһҪбһҡбһҠбҹ„бһҷбһӣбҹҒбһҒбһҹбҹҖбһ„** бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `iloc`бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ бҹҘ бһҮбһҪбһҡбһҠбҹҶбһ”бһјбһ„бһ–бһё DataFrameбҹ–
```python
df.iloc[:5]
```

**бһҖбҹ’бһҡбһ»бһҳ** бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ‘бһҪбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбһјбһ…бһҮбһ¶ *pivot tables* бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Excelбҹ” бһҹбһ“бҹ’бһҳбһҸбһҗбһ¶бһҷбһҫбһ„бһ…бһ„бҹӢбһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһҳбһ’бҹ’бһҷбһҳбһ“бҹғбһҮбһҪбһҡбһҲбһҡ `A` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһӮбһ…бҹҶбһ“бһҪбһ“ `LenB` бһ‘бһ¶бҹҶбһ„бһЎбһ¶бһҷбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ“бҹ„бҹҮвҖӢбһҷбһҫбһ„бһўбһ¶бһ…бһҖбҹ’бһҡбһ»бһҳ DataFrame бһҠбҹ„бһҷ `LenB` бһ бһҫбһҷбһ бҹ… `mean`бҹ–
```python
df.groupby(by='LenB')[['A','DivA']].mean()
```
бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҷбһҫбһ„бһ…бһ„бҹӢбһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһҳбһ’бҹ’бһҷбһҳ бһ“бһ·бһ„бһ…бҹҶбһ“бһҪбһ“бһ’бһ¶бһҸбһ»бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ»бһҳ бһҷбһҫбһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ `aggregate` бһҠбҹӮбһӣбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶бһ„бһ“бҹҒбҹҮбҹ–
```python
df.groupby(by='LenB') \
 .aggregate({ 'DivA' : len, 'A' : lambda x: x.mean() }) \
 .rename(columns={ 'DivA' : 'Count', 'A' : 'Mean'})
```
бһңбһ¶бһ”бһ„бҹ’бһ бһ¶бһүбһҸбһ¶бһҡбһ¶бһ„бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

### бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ


бһҷбһҫбһ„бһ”бһ¶бһ“бһғбһҫбһүбһҗбһ¶бһңбһ¶бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ Series бһ“бһ·бһ„ DataFrames бһ–бһёбһңбһҸбҹ’бһҗбһ» Pythonбҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶бһҳбһҖбһҖбҹ’бһ“бһ»бһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҜбһҖбһҹбһ¶бһҡбһўбһҸбҹ’бһҗбһ”бһ‘ бһ¬бһҸбһ¶бһҡбһ¶бһ„ Excelбҹ” бһ‘бҹҶбһҺбһ»бһҖбһ”бһүбҹ’бһ…бҹ„бһү Pandas бһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷвҖӢбһҷбһҫбһ„бһңбһ·бһ’бһёбһҹбһ¶бһҳбһүбҹ’бһүбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһҢбһёбһҹбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҖбһ¶бһҡбһўбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡ CSV бһӮбһәбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҠбһјбһ…бһ“бҹҒбҹҮ:
```python
df = pd.read_csv('file.csv')
```
бһҷбһҫбһ„бһ“бһ№бһ„бһғбһҫбһүбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹҖбһҸбһ“бҹғбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ–бһёбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ…бҹҒбһүбһҖбҹ’бһҡбҹ… бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖ "Challenge"


### бһҖбһ¶бһҡбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹ’бһҡбһ¶бһ”

бһўбҹ’бһ“бһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҠбһјбһ…бһ“бҹҒбҹҮбһңбһ¶бһҮбһ¶бһҡбһҝбһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҳбһҫбһӣбһғбһҫбһүбһңбһ¶бҹ” бһ“бҹ…бһ–бҹҒбһӣ DataFrame бһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһ’бҹҶ бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбһ„ бһҷбһҫбһ„бһ…бһ„бҹӢбһ’бҹ’бһңбһҫбһўбҹ„бһҷбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҷбһҫбһ„бһҖбҹҶбһ–бһ»бһ„бһ’бҹ’бһңбһҫбһўбҹ’бһңбһёбҹ—бһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһҠбҹ„бһҷбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ”бһүбҹ’бһҮбһјбһӣбһҮбһҪбһҡбһҠбҹҶбһ”бһјбһ„бҹ—бһҳбһҪбһҷбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” бһңбһ¶бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҠбҹ„бһҷбһ бҹ… `df.head()`бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһңбһ¶бһ–бһё Jupyter Notebook бһңбһ¶бһ“бһ№бһ„бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ– DataFrame бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбһ¶бһҡбһ¶бһ„бһҹбҹ’бһўбһ¶бһҸбһҳбһҪбһҷбҹ”

бһҷбһҫбһ„бһҖбҹҸбһ”бһ¶бһ“бһғбһҫбһүбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ `plot` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҹбҹ’бһҸбһ„бҹӢбһҖбҹ’бһ“бһ»бһ„бһҮбһҪбһҡбһҠбҹҒбһҖбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бҹ” бһҒбһҺбҹҲбһ–бҹҒбһӣ `plot` бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҖбҹ’бһҡбһ¶бһ”бһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бһҸбһ¶бһҳбһҡбһҷбҹҲбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ `kind=` бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ `matplotlib` бһҠбҹҶбһ”бһјбһ„ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶бһ„бһ“бҹҒбҹҮбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһ–бһёбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһӣбһҳбҹ’бһўбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһңбһӮбҹ’бһӮбһ”бҹҶбһ”бҹӮбһҖбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”

бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…бһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһ…бҹҶбһ“бһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһ”бҹҶбһ•бһ»бһҸбһҡбһ”бһҹбҹӢ Pandas бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ“бҹҒбҹҮбһҳбһ¶бһ“бһҳбһ¶бһҸбһ·бһҖбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ бһҫбһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҮбһ¶бһҳбһҪбһҷбһңбһ¶бһ‘бҹҒ! бһҷбһҫбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бҹ’бһҡбһҫбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ–бһ·бһҹбҹҒбһҹбһҳбһҪбһҷбҹ”

## рҹҡҖ бһ”бһүбҹ’бһ бһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ бҹЎбҹ– бһңбһ·бһ—бһ¶бһӮбһҖбһ¶бһҡбһ…бһҳбҹ’бһҡбһҫбһ“ COVID

бһ”бһүбҹ’бһ бһ¶бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„бһҠбҹӮбһӣбһҷбһҫбһ„бһ“бһ№бһ„бһ•бҹ’бһҸбҹ„бһҸбһӮбһәбһҖбһ¶бһҡбһӮбҹҶбһҡбһјбһ“бҹғбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бҹҮбһҮбҹҶбһ„бһәбһҡбһ¶бһҸбһҸбҹ’бһ”бһ¶бһҸ COVID-19бҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбҹҶбһ–бһёбһ…бҹҶбһ“бһҪбһ“бһўбҹ’бһ“бһҖбһҶбҹ’бһӣбһ„бһҮбҹҶбһ„бһәбһ“бҹ…бһ”бҹ’бһҡбһ‘бҹҒбһҹбһ“бһ¶бһ“бһ¶ бһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһҠбҹ„бһҷ [Center for Systems Science and Engineering](https://systems.jhu.edu/) (CSSE) бһ“бҹ… [бһҹбһ¶бһҖбһӣбһңбһ·бһ‘бҹ’бһҷбһ¶бһӣбҹҗбһҷ Johns Hopkins](https://jhu.edu/)бҹ” бһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһ¶бһ…бһҡбһҖбһ”бһ¶бһ“бһ“бҹ… [GitHub Repository бһ“бҹҒбҹҮ](https://github.com/CSSEGISandData/COVID-19)бҹ”

бһ–бҹ’бһҡбҹ„бҹҮбһҷбһҫбһ„бһ…бһ„бҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһңбһ·бһ’бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҷбһҫбһ„бһўбһүбҹ’бһҮбһҫбһүбһўбҹ’бһ“бһҖбһ”бһҫбһҖ [`notebook-covidspread.ipynb`](notebook-covidspread.ipynb) бһ бһҫбһҷбһўбһ¶бһ“бһ–бһёбһӣбһҫбһ‘бҹ…бһҖбҹ’бһҡбҹ„бһҳбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣбһҖбһјбһҠбһҖбҹ’бһ“бһ»бһ„бһҒбҹ’бһ‘бһ„бҹӢбһҖбһјбһҠ бһҖбҹҸбһҠбһјбһ…бһҮбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҮбһҪбһ”бһ”бҹ’бһҡбһ‘бҹҮбһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһ‘бһ»бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ“бҹ…бһ…бһ»бһ„бһ•бһ„бһҠбҹӮбһҡбҹ”

## бһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ·бһ“бһҡбҹҖбһ”бһҡбһҷ

бһҡбһҪбһҳбһ…бҹҶбһҺбҹӮбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҳбһ¶бһ“бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбһ¶бһҡбһ¶бһ„ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҒбҹ’бһӣбҹҮ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҳбһ·бһ“бһҡбҹҖбһ”бһ…бҹҶбһҡбһҪбһ… бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһўбһҸбҹ’бһҗбһ”бһ‘ бһ¬бһҡбһјбһ”бһ—бһ¶бһ–бҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ“бҹҒбҹҮ бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһҳбһҫбһӣбһғбһҫбһүбһҒбһ¶бһ„бһӣбһҫ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһўбһ¶бһ… **бһҠбһҖ** бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбҹҖбһ”бһ…бҹҶбһ”бһ¶бһ“бһҳбһҪбһҷбһңбһ·бһүбҹ” бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮбһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ–

* бһҠбһҖбһ–бһ¶бһҖбҹ’бһҷбһӮбһ“бҹ’бһӣбһ№бҹҮбһ–бһёбһўбһҸбҹ’бһҗбһ”бһ‘ бһ бһҫбһҷбһҳбһҫбһӣбһҗбһ¶бһ–бһ¶бһҖбҹ’бһҷбһӮбһ“бҹ’бһӣбһ№бҹҮбһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһ”бһ„бҹ’бһ бһ¶бһүй »зҺҮбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҠбһ„
* бһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ”бһҺбҹ’бһҸбһ¶бһүбһ”бҹ’бһҡбһҹбһ¶бһ‘ (neural networks) бһҠбһҫбһҳбҹ’бһ”бһёбһҠбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҡбһ”бһҹбҹӢбһңбһҸбҹ’бһҗбһ»бһ“бҹ…бһӣбһҫбһҡбһјбһ”бһ—бһ¶бһ–
* бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҳбһ“бһ»бһҹбҹ’бһҹбһ“бҹ…бһӣбһҫбһңбһёбһҠбҹҒбһўбһјбһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“

## рҹҡҖ бһ”бһүбҹ’бһ бһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ бҹўбҹ– бһңбһ·бһ—бһ¶бһӮбһўбһҸбҹ’бһҗбһ”бһ‘ COVID

бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ“бҹ’бһҸбһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҳбһ бһ¶бһ—бһ¶бһ– COVID бһ“бһ·бһ„бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһҸбҹ’бһҗбһ”бһ‘бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһўбҹҶбһ–бһёбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһ“бҹҒбҹҮбҹ” бһҳбһ¶бһ“ [бһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ CORD-19](https://www.kaggle.com/allen-institute-for-ai/CORD-19-research-challenge) бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ”бһ‘бһӣбһҫбһҹбһ–бһё 7000 (бһ“бҹ…бһ–бҹҒбһӣбһҹбһҡбһҹбҹҒбһҡ) бһўбҹҶбһ–бһё COVID бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһҳбһ¶бһ“бһҳбҹҒбһҸбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҹбһ„бҹ’бһҒбҹҒбһ” (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҳбһ¶бһҺбһҖбһ“бҹ’бһӣбҹӮбһ„бһ–бһ¶бһҖбҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣбһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ”бһ‘бһ–бҹҒбһүбһӣбҹҒбһүбһ•бҹ’бһҸбһӣбҹӢбһ‘бҹҖбһҸ)бҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ–бҹҒбһүбһӣбҹҒбһүбһ“бҹғбһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ“бҹғ [Text Analytics for Health](https://docs.microsoft.com/azure/cognitive-services/text-analytics/how-tos/text-analytics-for-health/?WT.mc_id=academic-77958-bethanycheum) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ–бһҺбҹҢбһ“бһ¶бһ“бҹ… [бһ”бҹ’бһҡбһҖбһ¶бһҹбһ”бҹ’бһӣбһҖбҹӢбһ“бҹҒбҹҮ](https://soshnikov.com/science/analyzing-medical-papers-with-azure-and-text-analytics-for-health/). бһҷбһҫбһ„бһ“бһ№бһ„бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһўбҹҶбһ–бһёбһҖбҹҶбһҺбһҸбҹӢбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҡбһҪбһҳбһ“бҹғбһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһ“бҹҒбҹҮбҹ”

> **бһ…бҹҶбһҺбһ¶бҹҶ**бҹ– бһҷбһҫбһ„бһҳбһ·бһ“бһ•бҹ’бһҸбһӣбҹӢбһ…бһҳбҹ’бһӣбһ„бһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһ‘бһёбһҸбһ¶бҹҶбһ„бһҖбҹҶбһҺбһҸбҹӢбһ“бҹҒбҹҮбһ‘бҹҒбҹ” бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҸбҹ’бһҡбһјбһңбһ‘бһ¶бһүбһҷбһҖбһҜбһҖбһҹбһ¶бһҡ [`metadata.csv`](https://www.kaggle.com/allen-institute-for-ai/CORD-19-research-challenge?select=metadata.csv) бһ–бһё [бһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӣбһҫ Kaggle бһ“бҹҒбҹҮ](https://www.kaggle.com/allen-institute-for-ai/CORD-19-research-challenge) бһҮбһ¶бһҳбһ»бһ“бҹ” бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҸбҹ’бһҡбһјбһңбһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһҮбһ¶бһҳбһҪбһҷ Kaggleбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ‘бһ¶бһүбһҷбһҖбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮ [бһ–бһёбһ‘бһёбһ“бҹҒбҹҮ](https://ai2-semanticscholar-cord-19.s3-us-west-2.amazonaws.com/historical_releases.html) бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһ“бһ№бһ„бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһўбһҸбҹ’бһҗбһ”бһ‘бһ–бҹҒбһүбһӣбҹҒбһүбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бһјбһҖбһҮбһ¶бһҳбһҪбһҷбһҜбһҖбһҹбһ¶бһҡбһҳбҹҒбһҸбһ¶бҹ”

бһ”бһҫбһҖ [`notebook-papers.ipynb`](notebook-papers.ipynb) бһ бһҫбһҷбһўбһ¶бһ“бһ–бһёбһӣбһҫбһ‘бҹ…бһҖбҹ’бһҡбҹ„бһҳбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣбһҖбһјбһҠбһҖбҹ’бһ“бһ»бһ„бһҒбҹ’бһ‘бһ„бҹӢбһҖбһјбһҠ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһ‘бһ»бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ“бҹ…бһ…бһ»бһ„бһ•бһ„бһҠбҹӮбһҡбҹ”
